This web page allows you to pair your Eight Sleep Pod 3, allowing it to connect to the desired Wi-Fi network. Normally this is accomplished through the Eight Sleep app, but due to persistent issues with their app (such as Bluetooth permissions being impossible to enable), I wrote this tool instead!

All code is executed locally in the browser. No information is transmitted, collected, or retained. Source code is open source and auditable. If you feel more comfortable, you can download and run this web page locally. This tool will also work with your device disconnected from Wi-Fi, if you want to be extra certain that no information is leaked.

Caveats

  • This tool works best with chrome://flags/#enable-web-bluetooth-new-permissions-backend set to Enabled, but isn't fully necessary. (Ignore this note if you aren't familiar with Chrome flags)

  • This tool was developed independently and is not affiliated with Eight Sleep

  • No warranty is implied while using this tool

‚úÖ Step 4: Confirm connection status

Confirm your pod has connected and successfully paired by looking at the LED color indicator:

  • White (pulsing or steady) - Success!
  • Yellow (pulsing) - Pod is unable to connect to Wi-Fi; maybe you supplied incorrect credentials? Trigger pairing mode (pulsing blue) and try again
  • Blue (pulsing) - Pod is still in pairing mode; please try these steps again
